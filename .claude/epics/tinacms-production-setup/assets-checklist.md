## Notas Técnicas

### Optimización de Imágenes
- **JPG:** Usar para fotografías, compresión 80-85%
- **PNG:** Usar para ilustraciones con transparencia
- **SVG:** Preferir para logos e iconos cuando sea posible
- **Tamaños:** Servir imágenes responsivas (considerar srcset)

### Estructura de Directorios Necesaria
```
public/images/
├── testimonials/     (crear - 11 imágenes)
├── products/         (crear - 4 imágenes)
├── activos/          (crear - 5 imágenes)
└── how-it-works/     (crear - 4 imágenes)
```

### Compatibilidad con Next.js
- Todas las imágenes deben optimizarse con Next.js Image component
- Considerar usar next/image para lazy loading automático
- Definir width/height explícitos para evitar layout shift
