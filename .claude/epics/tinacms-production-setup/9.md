# Task: Lighthouse audit y optimización

## Description
Ejecutar auditoría de Lighthouse y corregir issues para alcanzar score > 90 en performance.

## Acceptance Criteria
- [ ] Lighthouse Performance > 90
- [ ] LCP < 2.5s
- [ ] CLS < 0.1
- [ ] FID < 100ms
- [ ] Best Practices > 90

## Technical Details

### Ejecutar audit:
```bash
# En desarrollo
npx lighthouse http://localhost:3000 --output html --output-path ./lighthouse-report.html

# O usando Chrome DevTools
# F12 > Lighthouse > Analyze page load
```

### Áreas a revisar:
1. **Imágenes**: Verificar next/image, lazy loading, sizes
2. **JavaScript**: Bundle size, code splitting
3. **CSS**: Critical CSS, unused CSS
4. **Fonts**: Preload, font-display
5. **Third-party**: Defer non-critical scripts

### Fixes comunes:
- Agregar `priority` a imagen hero
- Configurar `sizes` en imágenes responsivas
- Verificar `loading="lazy"` en imágenes below-fold
