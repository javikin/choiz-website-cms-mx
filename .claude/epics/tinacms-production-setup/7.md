# Task: Configurar variables de entorno

## Description
Agregar variables de entorno de TinaCMS en desarrollo local y plataforma de hosting (Vercel/otro).

## Acceptance Criteria
- [ ] `.env.local` configurado con credenciales
- [ ] Variables configuradas en plataforma de hosting
- [ ] `.env.example` actualizado (sin secrets)
- [ ] Edición visual funciona en desarrollo

## Technical Details

### Archivo `.env.local`:
```bash
NEXT_PUBLIC_TINA_CLIENT_ID=<client-id>
TINA_TOKEN=<token>
NEXT_PUBLIC_TINA_BRANCH=main
```

### Archivo `.env.example`:
```bash
NEXT_PUBLIC_TINA_CLIENT_ID=your-client-id
TINA_TOKEN=your-token
NEXT_PUBLIC_TINA_BRANCH=main
```

### Plataforma de hosting:
1. Ir a Settings > Environment Variables
2. Agregar las 3 variables
3. Marcar como secrets (TINA_TOKEN)
